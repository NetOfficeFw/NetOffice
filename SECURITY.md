# Security Policy

## Supported Versions

NetOffice releases supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.9.6+  | :white_check_mark: |
| < 1.9.5 | :x:                |
| < 1.8   | :x:                |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities at <security@netoffice.io>
